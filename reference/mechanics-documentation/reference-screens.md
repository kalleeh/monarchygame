## Load/Save/Restore Screen

**Source**: https://www.monarchygame.com/boards/viewtopic.php?f=116&t=1732

### Overview
Game state management.

### Functions
- **Save**: Save current kingdom state
- **Load**: Load saved kingdom
- **Restore**: Enter restoration mode after death/destruction

### Restoration Mechanics
*(Detailed in strategy-guides/restoration.md)*
- **Damage-Based**: 48 hours (70%+ structure loss)
- **Death-Based**: 72 hours (complete elimination)
- Limitations during restoration period

## Leave NPP Screen

**Source**: https://www.monarchygame.com/boards/viewtopic.php?f=116&t=1731

### Overview
Exit New Player Protection status.

### Purpose
- Voluntarily leave protected status
- Become eligible for full gameplay
- Enable attacks on non-protected players

### Warning
Action is **irreversible** - cannot re-enter NPP once left.

These screens are primarily **UI/reference screens** rather than core gameplay mechanics. Implementation priority:

1. **High Priority**: Kingdom, UB Scouts, UB Wars, Scribe
2. **Medium Priority**: Scrolls, Scores, Race Info, Game Info
3. **Low Priority**: Banners, Prefs, Log Out
4. **Admin Functions**: Load/Save/Restore, Leave NPP, Suicide
