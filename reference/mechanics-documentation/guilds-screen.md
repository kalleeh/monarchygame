# Guilds Screen - Official Forum Documentation

**Source**: https://www.monarchygame.com/boards/viewtopic.php?f=116&t=1744

## Overview
Ranked and named alliances with larger groups of kingdoms.

A guild is a collection of kingdoms who unite under one banner to further their agenda as set out in the guild charter.

## Guild Purpose

### Typical Agendas
- Fight for their God
- Position themselves as powerful rulers
- Hire themselves out as mercenaries
- Each guild generally has different view on role and goals within Tonan

### Benefits of Joining
- **Protection**: Entitled to guild's protection
- **Training**: Greatest benefit - training provided by better guilds
- **Comradeship**: Community and coordination

## Guild Functions

### Viewing Guilds
- **'[Faithname] Guilds'**: Display guilds of your faith
- **'All Guilds'**: Display all current guilds
- **View Charter**: See guild's charter and requirements
- **Apply for Membership**: If guild is of your faith

### Rankings
- Display current guild rankings in game
- Show population of each guild

### War Declarations
- Display list of currently active war declarations between guilds
- **Note**: Not always accurate indication of ongoing combat

### Creating a Guild
- Submit guild charter for staff approval
- **Current Status**: All guilds automatically accepted - no wait required
- **Guild Password**: Set during creation for retrieval
- **Retrieve Guild**: Allows instant recreation of existing guild
- **Lost Password**: Use 'get lost guild pass' function to recover

### Contact Information
- Display emails and ICQ numbers for GM and Assistant GM
- Available for all currently active guilds

## Guild Management Screen

Once you join a guild, screen changes to Guild Management interface.

### Features
- Based around list of guild members
- View depends on assigned Privs and rank
- **Guide to Guild Privs**: Available in separate thread

### Guild Privs System
Each kingdom within guild has selection of privs granting various guild-related abilities.

### Member Viewing
- **'V' next to kingdom**: Show current state of that kingdom
- **'View All Guild'**: Show current state of all kingdoms in guild

### Application Management
- Alert when new applications received
- View basic details of applying kingdom
- Accept or reject applications

## Related Screens
- **Alliances**: Declare war between guilds
- **Kingdom**: View member kingdom details
